│                              🔄 DATA FLOW SEQUENCE                                    │
├──────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                      │
│  1. User selects teams      → Frontend calls /api/teams                              │
│                              → Backend loads teams.csv                               │
│                              → Data cached in LocalStorage                           │
│                                                                                      │
│  2. User selects players    → Frontend calls /api/players                            │
│                              → Backend loads squads.csv + players.csv                │
│                              → Player data cached in LocalStorage                    │
│                                                                                      │
│  3. User selects venue      → Frontend calls /api/venues                             │
│                              → Backend loads venues.csv                              │
│                              → Venue data cached in LocalStorage                     │
│                                                                                      │
│  4. User reviews summary    → Frontend loads from LocalStorage (instant!)            │
│                              → No API calls needed                                   │
│                              → Manual weather input                                  │
│                                                                                      │
│  5. User clicks predict     → Frontend sends POST /api/predict                       │
│                              → Backend loads all stat files                          │
│                              → Feature engineering (53 features)                     │
│                              → XGBoost prediction (70%)                              │
│                              → Neural Network prediction (30%)                       │
│                              → Ensemble combines both                                │
│                              → Returns: score, range, confidence, factors            │
